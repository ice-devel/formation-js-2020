Présentation JAVASCRIPT

Le javascript est un langage de programmation qui peut être exécuté directement dans un navigateur : il va permettre de dynamiser la page web

Le javascript est donc côté client : on executé des algos sans échange client / serveur.

Pour écrire du JS dans une page, il faut une balise script

On place en général le code javascript tout en bas du code de la page, car le chargement peut-être bloquant. On peut ainsi commencer à afficher le html/css alors que le javascript est en train de charger.
