Créez une page web avec deux champs texte, et un bouton valider. Créez le script Javascript qui réalise une multiplication entre les contenus des deux champs texte. Afficher le résultat dans la page web ou dans une fenêtre d'alerte. Si l'un des deux champs texte ne contient pas une valeur numérique au moment du clic sur le bouton valider, prévenir l'utilisateur de saisir une valeur numérique correcte.
