Créez une page web avec : - un champ texte - un bouton "enregistrer" - un bouton "afficher" - un bouton "supprimer" Créez un script javascript qui : - lors du clic sur le bouton "enregistrer", insère le contenu du champ texte dans une variable de type tableau. Chaque nouvelle entrée doit être placée à la fin de ce tableau. Le champ texte doit se vider automatiquement lorsque la valeur a bien été ajoutée dans ce tableau. - lors du clic sur le bouton "supprimer", cherche dans cette variable tableau si une valeur correspond au contenu du champ, et supprime cette valeur du tableau si elle existe. Sinon afficher un message précisant que la valeur n'existe pas dans le tableau. - lors du clic sur le bouton "afficher" : crée dynamiquement un élément div et l’insère dans la page où vous voulez affiche toutes les valeurs de la variable tableau dans ce nouvel élément HTML, avec une saut de ligne entre chaque valeur
